# אתר ניתוח התרעות - פיקוד העורף

אתר שמושך נתוני היסטוריה מפיקוד העורף ומציג:
- שיגורים
- כניסה למרחב מוגן
- יציאה מהמרחב המוגן

## מקור נתונים
- `https://www.oref.org.il/WarningMessages/alert/History/AlertsHistory.json`

## עקרון ספירה חשוב
כאשר בוחרים כמה יישובים יחד, אירוע שמופיע באותה שנייה ובאותה כותרת בכמה יישובים נספר כאירוע ייחודי אחד (דה-דופליקציה), ולא כהתרעות כפולות.

## יכולות
- סינון `כל השבוע` או `תאריכים + שעות`
- סינון לפי יישוב(ים)
- סינון לפי סוגי התרעות
- סינון לפי זמן שהיה בפועל במרחב המוגן (בדקות)
- השוואה בין כמה יישובים לאורך ימים
- טבלת אירועים ייחודיים + יצוא CSV
